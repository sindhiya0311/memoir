# 📖 Memoir

A modern full-stack digital journaling application built using the MERN Stack.

Memoir allows users to securely create, edit, organize, and manage personal journals with a clean and minimal interface.

---

**Intern ID:** CITS6160

## ✨ Features

- 🔐 User Authentication (JWT)
- 📝 Create Journals
- ✏️ Edit Journals
- 🗑 Delete Journals
- 🔍 Search Journals
- 😊 Mood Filtering
- 🏷 Tags Support
- 📚 Rich Text Editor
- 📱 Responsive UI
- 🎨 Modern Dark Theme

---

## 🛠 Tech Stack

### Frontend

- React.js
- Vite
- Tailwind CSS
- Framer Motion
- React Router
- Axios
- TipTap Editor
- Lucide Icons

### Backend

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT Authentication
- bcrypt
